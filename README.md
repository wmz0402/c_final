# 学生成绩管理系统  
***
## 功能特性
1. **用户认证模块**
    * **账户注册**：新用户注册有验证码校验。
    * **密码找回机制**：设置有安全问题，可通过回答设置安全问题来重置密码。
    * **安全机制**：
        * 密码输入时用`*`来掩饰。
        * 防暴力破解：连续输错4次密码将冻结账户10秒。
        * 密码强度校验：必须包含数字和字母。
    * **数据持久化**：用户信息保存在`user_date.txt`中。
    * **多角色登录**：分为管理员和普通用户两种身份进行登录

      
2. **学生成绩管理模块**
    * **数据结构**：使用单链表动态管理学生数据（尾插法）。
    * **功能详情**：
        * **录入**：添加学生信息（学号[^1]，姓名，系别，班级，高数，英语计算机）。
        * **显示**：支持打印全部，按系别打印，按班级打印。
        * **查询**：
            * 按学号精确查找到某一名学生。
            * 按照科目查询该科目的成绩表。
        * **修改**：按学号查找到要修改的学生，选择要修改的信息进行修改。
        * **删除**：按学号删除学生记录。
        * **排序**：支持按总分，高数，英语或计算机成绩进行排序。
        * **统计分析**：计算各科平均分，最高分，最低分，以及及格率/优秀率分析。
    * **数据持久化**：学生数据保存在`student_date.txt`中。
### 默认系统密码
在登陆成功进入主菜单后，选择进入具体系统是需要通过二级验证：  
| 系统类型 | 密码 | 权限 |
| -------- | --- | ---- |
| 管理员（admin) | `admin2007` | 拥有录入、修改、删除、排序等所有权限 |
| 学生系统（user) | `user0402` |仅拥有查看、查询、统计分析权限（只读） |
## 新手操作指南
1. **启动界面**：程序启动后会显示欢迎动画，随后进入下载/登录菜单。
2. **新用户**：选择`1`进行注册，设置保密问题。
3. **登录**：选择`2`登录，如果文件不存在，请先注册。
4. **选择系统**：
    * 输入`1`进入管理员系统（需输入`admin2007`）。
    * 输入`2`进入学生系统（需输入`user0402`）。
5. **退出**：在菜单中选择退出，会显示爱心字符画并结束程序。
### 环境要求
**编码格式**：建议源文件保存为**GBK**编码，以防出现中文乱码。
